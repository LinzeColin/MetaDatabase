## Scope

Added the Phase 11 operational trial evidence validator. The validator converts a
30-day production trial evidence package into the only accepted operational
evidence shape for production acceptance.

Validated evidence must prove:

- 30 unique daily trial dates.
- No duplicate source or publication IDs.
- P0 claims traceable for every daily run.
- No unsupported claims or misleading failure output published.
- Daily scheduler target time is `05:00`.
- Release, SMTP, RunRecord, and resource gate references exist per daily run.
- Scheduler is enabled with `04:45` health check and manual rerun evidence.
- Private Release or equivalent publishing evidence exists.
- Real SMTP delivery evidence targets `linzezhang35@gmail.com`.
- Disk, memory, cache, Git artifact, and secret hygiene evidence exists.
- Weekly replay, monthly replay, and failure recovery drill evidence exists.
