实现红线（本轮验证换来的教训，直接照做）：
1. `muted`/`loop` 必须以显式布尔 prop 或 ref 回调设置（`v.muted=true; v.loop=true; v.play().catch(()=>{})`）——裸属性在部分运行时不生效，未静音会被自动播放策略拦成冻结首帧。
2. `src` 不写在标签上，由 ref 回调按当前主题赋值——避免模板解析期对占位符发起无效请求（冷加载红错）。
3. 视频上必须压主题色渐变遮罩；浅色主题中段遮罩不低于 0.6 不透明度，正文不用 #6F6F6F 直压视频。
4. 生产化时应将三个视频转存自有存储（R2），CDN 直链仅原型期使用。

## 三种导航模式

- **sidebar**（暖纸/森林）：216px 左栏，分组标签 + 条目；暖纸=边线高亮直角，森林=胶囊高亮。
- **topbar**（简约专注/清新）：顶栏下方居中一行；专注=文字+激活底色 rgba(255,255,255,0.12)，清新=实底胶囊。
- **dock**（炫技/宇宙星河）：底部悬浮坞站 blur(18px)；炫技=999px 玻璃胶囊，宇宙=零圆角仪表条。
- 移动端（<780px）统一底部四标签（今天/队列/雷达/系统），主题只影响配色。
