# 00_只改这里

本页是 S2PIT01 的中文一改入口。它不是第二份配置文件，不保存配置值。

## 唯一编辑规则

- 只编辑：`config/owner_controls.yaml`
- 只读生成页：`docs/owner/OWNER_CONSOLE.md`、`docs/owner/SOURCE_CATALOG.md`、`docs/owner/MODEL_AND_QUEUE.md`、`docs/owner/CONTENT_LEDGER.csv`
- 修改后验证：`adp owner validate --controls config/owner_controls.yaml --json`
- 影响预览：`adp owner preview-impact --controls config/owner_controls.yaml --days 30 --json`

## 四类常用控制

| 中文控制 | 对应配置段 |
|---|---|
| 画像与目标 | `project`、`cost_policy`、`intelligence_provider`、`validation` |
| 邮件与复习 | `email`、`outputs`、`iteration` |
| 来源与板块 | `sources`、`boards`、`source_defaults` |
| 预算与调度 | `runtime`、`queue`、`scoring`、`validation` |

## 安全边界

本入口不能绕过 V7.2 gate，不能启用生产发件、自动调度、Release 上传、public schema 变更、DB migration、真实队列突变或 `INTEGRATED_PRODUCTION_ACCEPTED`。
