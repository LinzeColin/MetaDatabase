面包屑：`00_用户中心 / 当前状态`

本页是 `S2PMT06 / ACC-S2PMT06-UX` 的只读状态入口。它记录用户体验、安全操作、状态反馈、错误卡、配置变更预览、队列下钻、可访问性和不生产边界。

## 状态反馈

| 状态 | 含义 | 用户下一步 |
|---|---|---|
| `not_run` | 尚无本地证据收据 | 运行本地验证预览 |
| `loading` | 正在生成只读预览 | 等待或取消预览 |
| `no_update` | 本次无新可入选条目 | 查看未变化收据 |
| `partial_success` | 非关键板块已过期，但持久证据保留 | 打开降级说明区 |
| `degraded` | 非关键依赖不可用，正在显示兜底证据 | 打开可恢复错误卡 |
| `failed` | 必需本地验证门失败关闭 | 先看处置手册，再重试 |
| `stale` | 最新成功收据超过新鲜度阈值 | 运行安全刷新预览 |

不得用空表代替状态。

## 错误卡字段

每张阻断/错误卡必须包含这些机器字段：`code`、`severity`、`impact`、`owner`、`retry_safe`、`runbook`、`evidence`、`cta`。

## 安全操作链

用户可见配置和动作必须经过：`预览 -> diff/影响 -> 校验 -> 确认 -> 应用 -> 收据 -> 可回滚`。

适用动作：`retry`、`cancel`、`requeue`、`skip`、`regenerate`。本任务只生成本地收据，不写生产队列。
