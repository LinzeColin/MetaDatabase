| 能力 | 字段 |
|---|---|
| search | `cycle_id`、`mail_key`、`source_id`、`claim_id`、`status`、`owner_action` |
| filter | `mail_product`、`status`、`severity`、`source_domain`、`age_bucket`、`needs_owner` |
| sort | `created_at_desc`、`roi_desc`、`severity_desc`、`deadline_asc` |
| export | `json`、`csv` |
| drilldown | `queue_item -> source -> claim -> report -> mail -> review -> action -> roi` |
