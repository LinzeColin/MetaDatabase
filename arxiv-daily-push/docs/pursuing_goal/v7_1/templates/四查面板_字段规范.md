# 四查面板字段规范

所有面板必须有：generated_at、data_as_of、contract_version、contract_sha256、source_of_truth、freshness_status。

运行总控台必须按状态显示 current_count、today_added、oldest_age、timeout_count。

数据源健康必须显示 source_id、D、promotion_state、last_success、last_error、license_status、shadow_status、B1–B6 contribution。

模型队列必须显示参数和实际分布，不只显示 max capacity。

内容总账必须能从 content_id 追溯 Evidence→Insight→Mail→Review→Action→Asset→Conversion。
