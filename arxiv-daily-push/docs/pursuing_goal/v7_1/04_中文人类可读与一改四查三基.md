# 中文人类可读、一改四查三基与 GitHub 可验收要求

## 1. 用户第一原则

用户进入项目首先要完成的是：看状态、改设置、看队列、看内容、看复习和验收，而不是理解 `config/docs/governance/schemas/src` 的工程分类。

## 2. 推荐目录

```text
arxiv-daily-push/
├── README.md                         # 中文首屏入口
├── 00_用户中心/
│   ├── 00_开始这里.md
│   ├── 00_只改这里/
│   │   ├── 01_个人目标与画像.yaml
│   │   ├── 02_内容邮件与复习.yaml
│   │   ├── 03_数据源板块与权重.yaml
│   │   └── 04_运行预算与调度.yaml
│   ├── 01_运行任务与真实队列总控台.md
│   ├── 02_数据源与阅读板块健康.md
│   ├── 03_模型评分参数与全量队列.md
│   ├── 04_内容邮件复习行动与ROI总账.md
│   └── 05_系统总纲开发要求与验收标准.md
├── 功能清单
├── 开发记录
├── 模型参数文件
├── config/                           # 内部兼容配置
├── docs/governance/                  # 机器事实源
├── src/
└── tests/
```

“一改”指一个编辑目录，不要求把全部设置塞入一个巨型 YAML。

## 3. 四查

### 01 运行任务与真实队列总控台

首屏必须显示：

- 当前版本、Stage/Phase/Task、合同版本/哈希；
- 当前真实排队数量；
- 待证据、待分析、待讲解；
- 今日已完成讲解；
- 已生成报告；
- 待发送/已发送/发送失败；
- 今日复习到期、未来 7 天、逾期、已完成；
- 待行动、已完成实验、已形成资产、已记录转化；
- 每个状态的最老等待时间和超时数量；
- `generated_at`、`data_as_of` 和数据过期警告。

### 02 数据源与阅读板块健康

显示 D1–D4 每个来源的晋升状态、健康、最后成功时间、错误、许可、Shadow、回退和它贡献到 B1–B6 的数量。

### 03 模型评分参数与全量队列

显示：

- 所有评分模型和公式；
- 每个参数的中文名、ID、当前值、默认值、单位、范围、来源、影响、代码和测试；
- 实际队列分布、来源占比、板块占比、等待信用、淘汰原因；
- 参数影响预览和回滚值。

### 04 内容邮件复习行动与 ROI 总账

显示每项内容从发现到掌握的完整生命周期、证据级别、主讲状态、邮件状态、复习、行动、资产、预计 ROI 和实际转化。

## 4. 三基文件

### 功能清单

必须直接包含：

- V7 产品总纲；
- 4 源/3 主/3 副/3+1；
- 当前已实现、进行中、计划、阻断功能；
- 每项功能的价值、范围、非范围、代码、测试、证据、限制和验收；
- Owner 决策和下一唯一任务。

### 开发记录

必须直接包含：

- 完整 Stage→Phase→Task Roadmap；
- 每个 Stage 的 Pursuing Goal；
- 每个 Stage/Phase/Task 的 Stop Gate 和 Stop Conditions；
- 工时、状态、依赖、Acceptance、测试、证据、失败动作；
- 最近重要事件、当前阻断和回滚。

### 模型参数文件

必须直接包含：

- 所有活动模型、公式、假设和参数；
- 输入、输出、范围、缺失处理、回退；
- 当前值、默认值、先验、校准、限制；
- 代码、配置、测试和证据；
- 内容质量、路由、队列、复习、ROI、邮件调度、周/月报告参数。

三基文件不得是链接页或一句话跳转。

## 5. 全局中文优先

适用于 CodexProject 根仓库和全部注册项目：

- README 首屏、三基文件、用户中心、邮件、报告、PR 描述、CI 摘要和错误信息默认中文；
- 技术标识符、代码、路径、API、模型名和原始标题可保留英文；
- 首次出现的重要英文术语应有中文解释；
- 新增英文-only 人类界面触发 `LANGUAGE-UX-FAIL`；
- 自动生成器必须直接生成中文，而不是依赖人工翻译。

## 6. 唯一事实源与可读视图

- 机器事实源可以是 YAML/JSONL/数据库；
- 中文 Markdown 是完整的人类视图；
- 人类视图不得成为第二个可编辑事实源；
- 每个视图必须显示生成来源、合同版本、哈希、生成时间和数据时间；
- 漂移触发 `G-DRIFT`，不得进入下一任务。
