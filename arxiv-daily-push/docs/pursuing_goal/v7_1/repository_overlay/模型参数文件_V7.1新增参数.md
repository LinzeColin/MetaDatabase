# 模型参数文件｜V7.1 新增运行参数

| 参数 ID | 中文名称 | 默认/要求 | 影响 |
|---|---|---|---|
| runtime.lease_seconds | 运行租约秒数 | 必须配置并有上限 | 单实例与故障接管 |
| runtime.clock_skew_tolerance_seconds | 时钟漂移容差 | 明确值 | heartbeat/DST 安全 |
| runtime.shutdown_grace_seconds | 优雅关闭窗口 | 明确值 | drain/checkpoint |
| storage.busy_timeout_ms | SQLite 锁等待 | 明确值 | 并发稳定性 |
| storage.atomic_publish | 原子发布 | true | 半文件防护 |
| restore.allowed_root | 恢复允许根目录 | 唯一绝对根 | 路径穿越防护 |
| delivery.outbox_enabled | 事务发件箱 | true | 邮件重复恢复 |
| delivery.m4_deadline | M4 水位线截止 | 结构化时间 | 跨板块汇总 |
| scheduler.misfire_grace_seconds | 漏跑补跑窗口 | 明确值 | 休眠/DST |
| cache.max_bytes_by_class | 分类缓存上限 | 按类别 | 磁盘保护 |
| cache.ttl_by_class | 分类 TTL | 按类别 | 自动清理 |
| tests.soak_hours | 浸泡时长 | 生产候选≥24 | 稳定性证据 |
