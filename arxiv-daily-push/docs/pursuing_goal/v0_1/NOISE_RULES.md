# 模板噪声规则 · ADP-S2-P02-T026

`version_engine.strip_noise()` 在计算 `body_hash` **之前**清除展示层样板，使同一文档的纯装饰性重渲染
（页脚/导航/分享/计数/相对时间…）**不产生新版本**。规则**保守**：只匹配无歧义的样板，宁可漏删也不误删真实正文。

## 设计不变量

- **确定性**：纯正则 + 空白归一，无网络/时钟/随机；同输入 ⇒ 同输出。
- **不误删正文**：每条规则锚定行首样板 token；不做语义猜测。漏删（noise 残留）只会多留一点上下文，
  最坏是把一次纯噪声变化误判为实质变化（保守偏安全：宁可多留一个版本，也不吞掉真实修订）。
- **CJK 陷阱**：规则中**不得**在 CJK 备选词后接 `\b`。CJK 字符全部是 `\w`，`\b` 在两个 CJK 字符间永不触发
  （如 `分享到|微信`），会导致规则静默失配。改用 `token + .*` 锚定。（本任务因此发现并修复了一处真实缺陷。）

## 行级规则（整行命中即删）

| 类别 | 正则（大小写不敏感） | 例 |
|---|---|---|
| 版权/页脚 | `^\s*(?:版权所有\|copyright\|©\|\(c\)).*$` | `版权所有 © 2026 某网` |
| 编辑署名 | `^\s*(?:责任编辑\|编辑\|校对\|审核\|来源)\s*[:：].*$` | `责任编辑：李四` |
| ICP 备案 | `^\s*(?:京\|沪\|粤\|浙\|苏)?ICP备.*$` | `京ICP备12345号` |
| 分享/CTA | `^\s*(?:分享到\|扫一扫\|打开APP\|下载客户端\|阅读原文\|点击查看\|查看更多\|展开全文).*$` | `分享到微信` |
| 互动计数 | `^\s*(?:阅读\|浏览量?\|点赞\|在看\|评论\|转发)\s*[:：]?\s*\d[\d,\.万]*\s*$` | `阅读 12345` |
| 绝对时间戳 | `^\s*(?:发布于\|更新于\|编辑于\|发表于)\s*\d[\d年月日时分秒:：/\-\s]*$` | `发布于 2026-03-01 12:00` |
| 相对时间 | `^\s*(?:发布于\|更新于\|编辑于\|发表于)?\s*\d+\s*(?:秒\|分钟\|分\|小时\|时\|天\|周\|月\|年)前\s*$` | `发布于 3 分钟前` |

> **过删陷阱（已防）**：时间戳规则**必须要求「token + 前导数字 + 仅日期/时间字符」**，不能写成 `发布于.*[:：].*`——
> 否则会误删真正以「发布于」开头且含冒号的正文句（如 `发布于顶级期刊：Nature 的这项研究…`），**静默丢掉一次真实修订**。
> `t026_verify.py` 有专门的过删守卫用例。
| Cookie 横幅 | `^\s*(?:我们使用cookie\|本网站使用cookie\|接受所有cookie\|隐私政策\|cookie\s*settings).*$` | `我们使用cookie…` |
| 广告标记 | `^\s*(?:广告\|推广\|赞助内容\|sponsored\|advertisement)\s*$` | `广告` |
| 导航面包屑 | `^\s*(?:首页\|导航\|栏目\|频道\|menu\|nav)\s*[\|｜>》].*$` | `首页 > 科技` |

## 内联规则（行内替换为空）

| 类别 | 正则 | 例 |
|---|---|---|
| 追踪参数 | `[?&](?:utm_[a-z]+\|spm\|from\|src\|ref\|share_token)=[^\s&]+` | `…/a?utm_source=wx` → `…/a` |
| 内联相对时间 | `发布于\s*\d+\s*(?:分钟\|小时\|天)前` | 句中 `发布于 5 小时前` |

## 归一化

删噪后：`\r\n`→`\n`、多空白（含全角 `　`）折叠为单空格、去首尾空白、丢空行、`\n` 连接。

## 可扩展性

新站点样板出现时，向 `NOISE_LINE_PATTERNS` / `NOISE_INLINE_PATTERNS` 追加保守规则并补 fixture；
规则集不追求穷尽，只保证已知样板不再制造伪版本，且真实正文零误删（见 `known_gaps.md`）。
