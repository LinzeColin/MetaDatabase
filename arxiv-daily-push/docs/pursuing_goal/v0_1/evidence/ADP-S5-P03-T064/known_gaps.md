# Known gaps · ADP-S5-P03-T064

- **确定性标签抽取（非 LLM 语义抽取）**：method/sample/result 由**确定性标签/统计模式**抽取（`方法/method/approach:`、`样本/sample/dataset/被试/n=..`、`结果/结论/finding/conclusion:`）。**优点**：值恒为原文逐字节 span、**绝不幻觉**、可回原文、可重复。**代价**：覆盖率受标签形态限制——无标签、纯叙述式的方法/结果**报 missing（不猜）**而非语义抽取。语义级抽取由部署阶段/后续任务门控（并须保留本证据锚不变）。**这是有意的证据性取舍：宁可 missing 也不猜。**
- **首个 marker 命中优先**：`extract_fields` 取每字段**首个**匹配标签后到句界的 span。若同字段有「空值标签」后接「真值标签」（如 `方法：。方法：真实内容。`），取首个（空）→ 返回 missing 而非第二处真值——**保守报 missing、不猜**，不违反「缺失不猜」，但会**欠抽取**。真实论文罕见此形态；多标签合并留待后续。
- **标签需带冒号 + 精确边界（防误配/误截）**：标签规则要求 `\s*[:：]`，故 `methodology:` 不会误配 `method`（其后为 `ology` 非冒号）、`方法论：` 不会误配 `方法：`（其后为 `论` 非冒号）。**句界精确化**：CJK 终止符（。；）与换行恒为边界；**ASCII `.` 仅在句末（其后为空白或文末）才作边界**，故小数（如「准确率 92.5%」）**不被截断**（复核者指出的精度问题已修，见 verifier precision 控制）。**裸 `n=..` 统计**加负向前瞻 `(?<![A-Za-z])`，故 `functio[n] = 5` **不误配**（复核者指出的假阳已修），仅真正独立的 `n = 64` 兜底命中，值仍为原文 span。
- **句界/统计的残留启发式（诚实披露，非回归、恒可回原文）**：ASCII 句号即便精确化后，仍会在**英文缩写**（`e.g.`、`Fig. 3`、`vs.`）的 `. ` 处截断；裸 `n=` 守卫仅挡**前置 ASCII 字母**，故 `var_n=5`、`中n=5` 等仍会命中。二者**始终产出原文逐字节 span、绝不编造**（满足「回原文/缺失不猜」三条验收），且不弱于修复前行为；语义级缩写/统计消歧留待后续，本任务不需要。
- **筛选语义**：`filter_set` 支持 has-field（字段是否存在）与 keyword 子串（大小写不敏感）；确定性升序（按 paper_id）；无参调用返回全集（等价"不过滤"）。更丰富的布尔/范围/多字段筛选可扩展，但**可重复性**（确定性序）是硬约束。
- **comparison table 为库层数据**：逐篇 × method/sample/result，present cell 带证据定位、missing cell 带标记；渲染/交互视图由消费层（研究集合 UI）负责，本任务只出确定性数据。
