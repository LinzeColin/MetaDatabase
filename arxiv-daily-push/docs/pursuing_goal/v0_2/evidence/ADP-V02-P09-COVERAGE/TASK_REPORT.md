## ★核心决定：没有照搬 v0_1 的 gap_detector★
拿**发货的那个工具**对**真实线上数据**跑：

    unexplained: 0                 -> T043 验收 PASS
    source_not_yet_active: 3665/3937 = 93%

`infer_source_windows` 从**我们自己抓到的条目**反推源的起始月。我们最近才开抓，于是
arxiv-all 的 active_from 被推成 2026-07，2016 年那些月份全被标成「那时这个源还不存在」——
**对 arXiv（1991）、Nature（1869）、NEJM（1812）、PNAS（1915）、bioRxiv（2013）这就是假话**，
仅这 8 个源就 **1003 格**。

**那条验收是靠一个假解释通过的**，而它**分辨不出「诚实的空洞」和「错的解释」**。错的解释比空洞更糟。
故本次**不引入** `source_not_yet_active`：没有独立证据证明某源当时不存在，就不许这么说。
默认 `not_backfilled` 是**可证的事实**。页面呈现的不是「0 空洞、很全面」，而是**真实的覆盖债务**。
