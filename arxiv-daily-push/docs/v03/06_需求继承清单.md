- **六个一级入口**：今天学什么 / 学习队列 / 前沿雷达 / 证据与纠错 / 应用与资产 / 系统与来源 —— 原型已按此实现。
- **三大核心旅程**：每日专注（J1）/ 纠错传播（J2）/ 权重修改提案→预览→应用（J3）—— 分别落在 R1 / R2 / R4，原型均有对应界面。
- **五类声明分型**（事实/作者主张/推断/假设/行动）—— 原型证据表已用「论文事实/作者主张/边界声明」呈现，完整五型在 R2 落库。
- **学习状态机**（未见→接触→投入→回忆→保留→应用→掌握，含过期/重开）—— 队列状态编辑与纠错重开覆盖。
- **手动状态编辑**（你的自定义要求）—— 每篇可手动改「未学习/已学会/需复习/已掌握」，记录为主观标记、可撤销、与回忆证据分开统计。
- **发布五步授权**（实现→验收→受控试运行→持久授权→续期/撤销）—— 原型授权卡 + R4 第四周五决策。
- **板块一～五**（研究前沿/顶级期刊/中国政策法规/美国科技金融/跨板块总览）—— 雷达页板块切换 + 来源权重区，B2–B4 走启用提案。
- **评分维度归并说明**：你原设计的中国政策 8 维、美国官方 8 维评分体系未删除——作为板块专属特征集预留在阈值注册表的扩展区，待对应板块启用时激活（先跑通，再扩维）。
