# 用户手册（Owner 日常使用）

打开方式：本机 `http://127.0.0.1:8787`；部署 R6 后手机可用 `home.linzezhang.com`（Cloudflare Access 登录，仅你可见）。

## 每天 10 分钟

1. **今天学什么**：先看「为什么今天选它」（含为什么不是第二名）；读讲义（约 12 分钟，八个部分，句句可溯源）。
2. **主动回忆**：读完先在心里复述 → 按住显影区 0.8 秒看答案 → 按诚实程度点四档之一（忘了/困难/良好/轻松，按钮上直接写着下次复习时间）。**只有这个动作能产生「已学会」。**
3. **今日复习**：进「学习队列」清掉到期项（通常 3–6 分钟）。
4. 想更深：点「打开 ChatGPT 新对话」——会带上论文元数据、你已学要点和你的深挖指令，不用重复解释背景。

## 你的控制权

- **手动改状态**：任何一篇都能改成 未学习/学习中/已学会/需复习/已掌握；再点一次撤销。手动标记与回忆证据分开统计，纠错提醒不受豁免。
- **手动调参数**：系统与来源 → 「打开参数风洞」拖动权重先看效果 → 满意后「发起提案」→ 预览 30 天回放差异 → 应用（有回执，可回滚）。
- **自迭代**：每周日收到一份权重微调提案（基于你本周的回忆成绩、跳过与弃权），一键应用或驳回；每月一号画像更新；回忆满 100 条后系统训练你的专属 FSRS-6 参数。
- **主题**：右上角切换六种主题，即时生效并记住。
- **弃权日**：收不到讲义说明当天最高分低于弃权线——不打扰是特性；候选没有丢，都在池子里。

## 数据安全（任何数据不会丢）

所有事件只增不删（回忆、评分、手动标记、调参、发送，包括失败）；每日自动备份、每周异地快照（R6 后镜像到你的 Cloudflare D1+R2）；「系统与来源 → 数据永不丢失」卡片里可一键导出全部数据。

## 出问题时

看「系统与来源」最近运行行：正常/降级/弃权/失败/未运行，每行都是人话。仍不明白 → `05_运维手册/排错手册.md` 按症状查。
