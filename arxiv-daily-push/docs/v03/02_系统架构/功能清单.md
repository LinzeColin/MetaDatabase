# 功能清单（按六域组织）

标注：✅ 现有保留 · ♻ 重构 · ➕ 新增 · 🔁 替代为成熟组件 · ⏸ 冻结

## 发现域
- ✅ arXiv 官方接口接入（已验收，30 天回放证据）
- ➕ 来源政策快照（接口条款/限速/核验日期版本化）
- 🔁 板块二～四统一订阅面（RSSHub 自托管 + feedparser，新来源=一行配置）
- ➕ 去重/版本链 · 来源健康事件 · 连续 3 天失败自动停用

## 证据域
- ➕ 证据声明落库（分型：论文事实/作者主张/推断/假设/行动，含原文定位）
- 🔁 元数据增强（OpenAlex `pyalex` + Semantic Scholar 官方客户端，免费层，失败只降级不阻塞）
- ➕ 版本/撤稿检测 → 纠错事件 → 受影响知识自动重开 + 强提醒
- ➕ 矛盾登记与 Owner 裁决界面

## 选择域
- ♻ 两段式选择：5 资格硬门（布尔）→ 8 特征加权（相关22·知识差距20·相对新颖14·迁移12·遗忘8·紧迫6·证据5·多样性17）− 注意力成本扣分
- ➕ 弃权机制（最高分 < 55 当日不发，理由进周报）
- ➕ 双向解释：为什么选它 / 为什么不是第二名（含特征贡献）
- ✅ 候选池 ≤10,000 · 单源占比 ≤40%（继承现值）

## 学习域
- ➕ 深度讲义（八段结构，目标系统内学到 80–90%，逐句溯源校验）
- 🔁 讲义生成：Codex 认证通道（沿用现有 intelligence_provider 合同）；失败降级确定性模板
- ➕ 主动回忆（按住显影 + 四档自评）→ 学习事件
- 🔁 复习排程：FSRS-6（py-fsrs 6.3.1，21 参数官方默认；≥100 条回忆后个性化训练）
- ➕ 手动状态编辑（主观标记单独记账、可撤销）
- ➕ 深挖直达：携带全文上下文 + Owner 深挖指令打开 ChatGPT 新对话

## 应用域
- ➕ 迁移练习（绑定 Owner 真实项目）· 知识资产（笔记/清单/决策记录，带来源）· 结果记录（失败也记）→ 反哺排序

## 交付与运行域
- ♻ 网页应用（六入口、六主题、桌面+移动、声音/触感反馈）
- ♻ 邮件镜像（与网页同源渲染、幂等防重、发送授权凭证可过期可撤销）
- ➕ 每周雷达 · 每月前沿地图 · 知识债务账本
- 🔁 调度：launchd 定时 + 心跳 + 睡眠补跑（删除自研守护 ≈70KB）
- ➕ 运行记录五态（正常/降级/弃权/失败/未运行）→ 自动生成用户中心
- ➕ 参数自迭代（周提案/月画像）与前端手动调参（提案→预览→应用→回执）
- ➕ 数据永不丢：事件只增不删、每日备份、每周快照、一键导出
- ➕（可选 R6）Cloudflare 混合部署：Pages 前端 + Access 私登 + D1 镜像 + R2 快照 + Worker 提醒
- ⏸ 冻结：视频/语音/分镜命令、Release 媒体交付、362KB 单体命令行中与闭环无关的命令
