## 三、报错了怎么办

| 遇到什么 | 为什么 | 怎么解决 |
|---|---|---|
| 探针缺失 | 工作流未执行或证据未落盘 | 状态记为 UNKNOWN，禁止部署或继续稳定期 |
| 免费容量接近上限 | 请求、计算、cron 或存储预算不足 | 停止扩张并生成付费升级提案，不自动消费 |
| 生产阻断检查失败 | 构建、来源、数据或界面不满足锁定 Oracle | 立即回滚到最近接受版本并重新开始稳定序列 |
| 历史 ZIP 出现真实凭据 | 旧证据包含未识别 secret | 停止公开分发、轮换凭据并执行 Git 清史 |
