# 路线图与停止门

更新时间：2026-07-01 15:00:20 Australia/Sydney

本页说明 ADP 当前在哪个阶段、什么能继续、什么必须停止。

## 当前路线图位置

| 项目 | 当前口径 | 中文解释 |
|---|---|---|
| 当前产品合同 | V7.2 | 以后判断 Stage 2 时以 V7.2 为准 |
| V7.1 | 只读历史基线 | 不能覆盖或删除，只能继承有效要求 |
| 当前阶段 | Stage 2 | 多来源、多邮件、多复习行动收益链路仍在推进 |
| 当前最终门 | S2PMT07 | 这是正式生产前的最终预检查 |
| 当前结论 | Final bundle artifact chain 已收口；生产验收仍未通过 | 可以审查最终包并进入生产验收边界预检，但还不能宣称正式每日生产通过 |

## 停止门

| 停止门 | 当前状态 | 没通过前禁止什么 |
|---|---|---|
| Stage 2 正式生产验收 | 未通过 | 禁止说多来源生产已完成 |
| 历史问题清零 | 当前 zero-proof 为一级 0 个、二级 0 个；V7.1 baseline 一级 8 个、二级 37 个保留为历史基线 | 禁止覆盖或删除历史 baseline；禁止把 zero-proof 直接扩展成生产运行开关 |
| 集成候选完成 | Final bundle 已通过；生产边界未通过 | 禁止说 DAILY_OPERATION 已就绪 |
| 独立最终复审 | 已有独立 signoff artifact；不等于生产验收 | 禁止把 signoff 扩展成正式运行开关 |
| 最终验收包 | 已完成并通过 validator；`missing_items=[]` | 禁止把 final bundle pass 解读为 `INTEGRATED_PRODUCTION_ACCEPTED` 或 `DAILY_OPERATION` |

## 可以继续的工作

| 工作 | 是否可以继续 | 条件 |
|---|---|---|
| 修复用户中心中文可读性 | 可以 | 不改生产代码和公共结构 |
| 补齐复习行动收益每日快照 | 可以 | 只做状态展示，不宣称生产验收 |
| 无冲突的影子数据源证据 | S2PMT07 阻断期暂停新增影子数据源 | 仅维护已完成历史证据；不得把新增影子来源或板块作为当前下一任务 |
| M1 到 M4 邮件模板一致性检查 | 可以 | 不绕过 Email V1 合同 |

## 禁止越界

| 禁止项 | 原因 |
|---|---|
| 把一次补发成功说成正式生产通过 | 邮件成功只是局部事实 |
| 把本机运行器说成 GitHub 云端生产 | 当前生产策略不是 GitHub 云端每日运行 |
| 没有独立复审就清零历史问题 | 违反 V7.2 最终门 |
| 没有用户中心数量快照就说体验完整 | 用户仍看不到每日复习和收益数量 |
