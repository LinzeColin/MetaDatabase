## 口径定义

- 已发送：SMTP report 是 `sent`，并且 ledger 里有发送引用。
- 未发送：运行状态出现 `blocked`、`failed`、`dry_run`，或真实 SMTP 为 `false`。
- 排队：候选项仍是 `queued`，表示还在池子里，未作为本次邮件主论文发出。

## 证据边界

- 这页是 GitHub 人类可读快照，不是运行系统本身。
- 本机 JSON、SMTP report、ledger 和 candidate queue 是底层证据来源；owner 不需要直接打开这些本机文件。
- Gmail 页面是否落在 Inbox、Spam、All Mail 需要邮箱侧确认；当前系统证据只证明 SMTP 已接受发送。
