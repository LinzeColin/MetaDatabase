| 每次真实发送、补发、阻断或跳过 | 本页“总览”和“历史发送记录” |
| 每次候选池变化 | 本页“总览”和“截至今日候选队列”，以及 [用户中心首页](./README.md) |
| 每次邮件模板升级 | 本页“邮件发送模板”、[邮件模板预览](./邮件模板预览.md)、[一看三查](./一看三查.md)、[关键结论与用户决策](./关键结论与用户决策.md)、三基文件和运行清单 |
| 每次页面内容更新 | 运行 `python arxiv-daily-push/scripts/update_user_center_timestamps.py --write` 写入真实更新时间，再运行 `--check` 拦截未来时间 |

更新目的地固定为 GitHub：`arxiv-daily-push/用户中心/邮件发送与队列状态.md` 和 `arxiv-daily-push/用户中心/邮件模板预览.md`。本机运行文件、SMTP 报告和 JSON 只能作为证据来源，不能替代 GitHub 用户中心页面。
