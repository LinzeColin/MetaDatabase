## 阅读规则

- GitHub 页面是 owner 人类可读主界面。
- 本机运行目录只作为证据来源，不作为 owner 阅读入口。
- 如果 GitHub 页面和本机 JSON 不一致，先停止扩张，按证据刷新 GitHub 页面，再继续判断。
- 三基文件必须是中文完整内容，不得退化成跳转页。
- 不把 owner 文档当作生产开关；补发、改队列、改 SMTP、改 scheduler、改 schema 都必须走明确任务和验证。
