## 收益口径

| 类型 | 可以怎么写 | 禁止怎么写 |
|---|---|---|
| 预期收益 | 基于假设、置信度、行动窗口给出估计 | 写成确定收益 |
| 实际收益 | 有成本、产出和证据时再记录 | 没有证据就填数字 |
| 能力收益 | 记录可复用方法、模板、判断力提升 | 写成空泛成长总结 |
| 每日数量 | 从当日运行报告同步到 GitHub 用户中心 | 用测试 fixture、旧报告、聊天内容或人工猜数代替 |
