## 容易误判的地方

| 看到的事实 | 正确解释 | 错误解释 |
|---|---|---|
| 受控发送证据 / 计划应发有进度 | 邮件发送有真实记录；计划应发为 M1-M4 每日 3+1 计划目标 | 整个 Stage 2 已经生产通过 |
| Final bundle artifact chain 已收口 | 最终验收包可审查，并已支持 Stage 2 integrated acceptance；真实发送开关和后台任务仍保持关闭 | 已经可以启用每日后台运行或声明 S3/DAILY_OPERATION 已进入 |
| 总候选池大于前20精选 | 总候选池 299 条；候选队列前20精选只是按评分抽取的阅读入口 | 系统只剩 20 条候选，或 20 条就是总候选队列 |
| 数据源清单显示 6 个来源 | 当前只有 arXiv 来源生产启用；其他来源是影子测试或规划 | Stage 2 多来源已经全部生产通过 |
| 本机计划任务曾配置或存在 | 本机运行方案只是历史/受控运行证据来源；当前 `daily_operation_enabled=false` 且 LaunchAgents disabled | GitHub 云端或本机计划任务已经成为持久每日生产运行器 |
| 一次受控真实运行验收已通过 | 只说明受控窗口内的发送证据；窗口结束后仍要恢复 `UNSET` 或 false-like，并保持 enablement preflight fail-closed | 持久 DAILY_OPERATION 已授权，或可以留下 truthy SMTP 开关 |
| 复习行动收益有本地证据 | GitHub 已展示字段、证据链和 2026-06-28 服务日快照数字；后续每日仍要由运行快照同步 | 可以人工猜数字、拿测试样例当今天结果，或宣称持久在线日报已验收 |

计划来源：Email V1 每日 3+1（M1, M2, M3, M4），计划应发 4 封；受控发送证据不代表 S3/DAILY_OPERATION 已进入。
