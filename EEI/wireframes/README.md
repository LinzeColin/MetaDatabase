# Wireframe Handoff Notes

这些文件是 **v4.2 已批准的可视化工作台基线**，用于 Codex 视觉回归与交互验收，不是旧版行业卡片首页。

## 文件

- `home_default_view.svg/png`：默认首屏。左侧导航与 Watchlist 固定存在，中间直接展示当前公司的关系图，右侧为人类可读摘要，底部为时间变化。
- `recursive_focus_explorer.svg/png`：从 NVIDIA 点击 TSMC 并选择“以它为中心”后的主体切换状态。
- `visual_first_workspace.svg/png`：首屏结构的规范入口。
- `visual_first_workspace_screenshot.png`：浏览器实际渲染的 1440×900 回归基准。
- `dashboard_wireframe.svg/png`：为旧文件引用保留的兼容名称，内容已指向新版工作台，不得据此恢复卡片式首页。

## 核心交互不变量

1. 默认入口是 Watchlist-first、map-first；行业是左侧入口与空状态，不是占据首屏的卡片墙。
2. 关系必须优先通过位置、方向箭头、连接线、关系标签、分组边界和状态样式表达；文字用于解释和核验。
3. 上游默认在左、当前主体居中、下游/客户在右；业务板块在上；资本、政策和风险位于下方或相邻稳定区域。
4. 单击节点只选中并更新右侧详情；只有明确点击“以它为中心”才切换主体并重绘版图。
5. 主体切换继承截至日期、视角、筛选、缩放、时间范围和分析偏好；breadcrumb、返回与保存视图均可恢复。
6. 首屏避免“毛线团”：默认显示最重要的 20–40 条关系，其余按阶段、行业、关系族聚合并按需展开。
7. 所有图形信息必须有列表替代、键盘路径和非颜色唯一编码；关系证据可追溯。
8. 用户可查看并调整权重、阈值和时间衰减；变更进入操作日志，模型每 14 天复核一次，建议不自动生效。
