# Atlas v4.2 可交互原型

- `standalone.html`：无需服务即可打开的单文件原型。
- `index.html`：与 standalone 完全一致，便于静态托管。
- `styles.css` / `app.js`：供 Codex 拆分生产实现时参考。
- `screenshots/`：浏览器回归截图。

## 数据真实性边界

当前原型使用示例快照和 fixture 关系，目的仅是验证信息架构、可视化、递归切换、参数编辑、全局刷新反馈和治理导航；不代表真实数据库已接入，也不构成投资建议。生产实现必须依据来源、证据与时间字段逐条生成关系。

## 操作路径

1. 首页直接进入 Watchlist 当前公司商业版图。
2. 点击节点打开证据抽屉，再点击“设为研究中心”递归切换。
3. 在“对象与范围”查看关系、供应链、行业、公司、业务与资本目录。
4. 在“模型与参数”修改权重/阈值，预览影响，再发布以触发全局刷新演示。
5. 在“开发治理”查看功能、任务、风险、验收与生产边界。
