# Sources

## UI, accessibility and motion

机器清单：`data/uiux_benchmark_sources.csv`。主要基准包括 W3C WCAG 2.2、Material Design 3 motion、Apple HIG、复杂图谱产品和数据库工作台模式。

## GitHub governance

- GitHub Issue Forms：结构化收集功能、模型和数据变更。
- Pull Request Template：要求范围、测试、风险、回滚和文档同步。
- CODEOWNERS：目录变更自动请求责任人审查。
- GitHub Actions path filters：在相关目录变化时运行一致性校验。
