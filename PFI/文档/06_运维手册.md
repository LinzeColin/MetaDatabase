## 一、怎么跑


## 三、报错了怎么办

| 遇到什么 | 为什么 | 怎么解决 |
|---|---|---|
| 页面显示来源未加载 | 余额、负债、持仓、价格或汇率快照缺失 | 从来源中心登记真实来源；不要手填零值绕过。 |
| 发布身份冲突 | App、manifest、前后端 hash 或运行响应不一致 | 停止运行并重新构建同一候选；不要在最终验收时临时补丁。 |
| 数据库恢复被拒绝 | 目标 hash、sidecar、权限、文件系统或备份校验不满足 | 先在隔离目标验证，保留原库并重新取得一致备份。 |
| 历史真实源在当前树中不存在 | 上游已迁移顶层 MetaDatabase | 使用 v0.2.5 immutable source lock 读取已复核 Git 对象；不得恢复旧目录。 |
