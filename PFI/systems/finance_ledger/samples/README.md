# Sanitized Finance Ledger Samples

These files are synthetic samples for parser and onboarding tests. They do not contain user transactions, account identifiers, merchant order ids, or real source file names.

Files:

- `sanitized_alipay.csv`: minimal Alipay-style bill with three synthetic rows.
- `sanitized_wechat.csv`: minimal WeChat-style bill with three synthetic rows.

Use them only for smoke tests, demos, and schema examples. Real bills must stay outside GitHub under local private data directories.
