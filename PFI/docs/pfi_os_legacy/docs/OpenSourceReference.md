# Open Source Reference

## Current Reference Set

PFIOS 当前参考的开源方向包括 QuantStats、Qlib、Backtrader / cloudQuant Backtrader、vectorbt / pyfolio 风格 tear sheet，以及 Monte Carlo dashboard 类项目。

PFIOS currently references QuantStats, Qlib, Backtrader / cloudQuant Backtrader, vectorbt / pyfolio-style tear sheets, and Monte Carlo dashboard projects.

## Adopted In This Build

QuantStats / pyfolio 方向：回测 Word 报告新增收益对比、月度收益热力图、滚动夏普、滚动波动率、交易回合质量和成本压力诊断。

QuantStats / pyfolio direction: Word backtest reports now include return comparison, monthly return heatmap, rolling Sharpe, rolling volatility, round-trip trade quality, and cost-stress diagnostics.

Backtrader / vectorbt 方向：网页工作台保留交互式 K 线、买卖点、收益曲线、回撤、交易摩擦和交易明细。

Backtrader / vectorbt direction: the web workbench keeps interactive candlestick, buy/sell markers, returns, drawdown, trading friction, and trade detail views.

Qlib 方向：研究闭环继续强调数据处理、回测、风险建模、成本、组合归因、市场环境分层和可追溯报告。

Qlib direction: the research loop continues to emphasize data processing, backtesting, risk modeling, costs, portfolio attribution, market-regime breakdown, and traceable reports.

Monte Carlo dashboard 方向：后续优先加入参数鲁棒性分布、最大回撤分布和情景模拟。

Monte Carlo dashboard direction: next priorities are parameter robustness distributions, max-drawdown distributions, and scenario simulation.
