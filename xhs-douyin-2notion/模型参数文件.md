# 模型参数文件

当前阶段不配置或调用任何 AI Provider。

| 参数 | 当前值 | 门禁 |
|---|---|---|
| `provider` | `disabled` | Owner 显式授权后才能变更 |
| `cloud_enabled` | `false` | 私人内容外发评审通过后才能开启 |
| `budget` | `0` | 必须给出币种、周期和硬限额 |
| `prompt_version` | `UNSET` | Stage 4 前不得伪造 |
| `model_id` | `UNSET` | 必须 Pin 且记录 Provider/版本 |
| `taxonomy_mutation` | `forbidden` | AI 永不得创建一级分类 |
| `evidence_required` | `true` | ASR/OCR/Vision 输出必须可追溯 |
| `eval_dataset` | `x2n-synthetic-model-contract-v1@1.0.0` | 仅验证 Dataset Contract，不调用模型 |
| `asr_enabled` | `false` | 对应能力 Acceptance 通过后才可开启 |
| `ocr_enabled` | `false` | 对应能力 Acceptance 通过后才可开启 |
| `fusion_enabled` | `false` | 对应能力 Acceptance 通过后才可开启 |
| `classification_enabled` | `false` | 自动分类还必须等待 `ACC.x2n.ai.006=PASS` |

Secret、API Key、Token、私人 Prompt/Response 和本机绝对路径禁止写入此文件或 Git。
