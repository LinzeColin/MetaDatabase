# 10 README Requirements

The final project README must include:

1. What the system does.
2. What the system does not do.
3. Safety boundary: no automatic trading.
4. Setup commands.
5. Alipay CSV template format.
6. How to run one slot manually.
7. How to inspect the latest report.
8. How to test notification dry-run.
9. How scoring works.
10. How benchmark comparison works.
11. How risk hard gates work.
12. How fallback and data quality downgrades work.
13. How to schedule with launchd.
14. Troubleshooting:
    - moomoo/OpenD unavailable.
    - Apple Mail not configured.
    - missing official fund source.
    - stale NAV.
    - malformed holdings CSV.

Required README warning:

```text
This tool produces research, ranking, discipline labels, and notification drafts. It does not place trades. Future outperformance versus Shanghai Composite or S&P 500 cannot be guaranteed.
```
