# 模型参数文件

## 摘要

- project_id: `MetaDatabase`
- product_version: `0.1.0`
- active_model_count: `0`
- active_formula_count: `0`
- active_parameter_count: `0`

## 当前说明

MetaDatabase 是数据归档项目，不包含业务模型、公式、评分权重、阈值或推荐参数。

| 类型 | 数量 | 说明 |
| --- | --- | --- |
| active models | 0 | 不做分析或决策 |
| active formulas | 0 | 不计算指标 |
| active parameters | 0 | 不设置阈值 |

## 边界

- 数据解析、分类、阈值和建议属于业务系统，如 `PFI/`。
- MetaDatabase 只保存原始数据、处理后文件和 manifest 证据。
